# Project Idea
- Bullet Hell game inspired by Touhou

# Controls
- W to move up
- A to move left
- S to move back
- D to move right
- Space to shoot
- Shift to move slower
- V to activate the character's ability

# Current Content
- 3 Levels
- 9 Enemies
- 3 Bosses
- 3 Characters with different abilities
